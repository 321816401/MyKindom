# imking启动大会

## 红包需求讲解
 * 1对1发红包方式(同一时间一人抢答红包)，遵循契约精神
 * 红包问题默认有效期是3天，红包问题无人抢答的时候可以撤销，或者增加金额继续开放
 * 发红包时没有足够干粮的时候，可以充值再回来继续发，也可以发免费红包问题，但每人一天只允许发一次免费红包问题
 * 红包问题太复杂的情况，可以申请延期，最多3天
 * 红包刷新即更改发布时间，提升列表首页的排名
 * 红包问题部分解决的情况，支付部分金额，未解答的部分，可以继续发接龙红包或关闭
 * 红包支付金额，需要被支付者确认操作
 * 详细内容可查看剪影发的需求原型和文档
